# Security

This repository is a demo reference implementation. Do not report real API
keys, webhook secrets, personal data, payment data, or production customer
identifiers in a public issue.

Report vulnerabilities privately to `security@yario.ai`. Include the affected
commit and a minimal reproduction with synthetic data only.

The application refuses live credentials by default and requires explicit
installation and client allowlists. These controls must not be removed when
adapting the example.
